1. для дробных чисел, необходимо использовать разделитель точка
2. поддерживаются только круглые скобки
3. поддерживает арифметические функции +, -, *, /, -(унарный)
4. поддерживает логические функции <, >, ==, !=, <=, >=
5. поддерживает условный тернарный оператор ?:
5. логические функции можно применять только в условии тернарного оператора (подразумевается, что он возвращает только число)
6. реализована базовая проверка синтаксиса:
    - наличие недопустимых символов (допустимыми являются числа, пробел, арифметические функции, логические функции, условный тернарный оператор, скобки)
    - проверка сочетания скобок
    - проверка сочетания тернарных операторов
7. если используется вложенный тернарный оператор, его следует выделять скобками
